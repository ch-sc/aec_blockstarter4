# Blockstarter 4.0

## Prerequesites

* node >= 7.0.0
* npm

## Install

```
npm install -g ethereumjs-testrpc truffle
```

```
cd api/ && npm install
```

```
cd client/ && npm install
```
```
cd cli/ && npm install
```


## Running the application

Start testrpc:
```
testrpc
```

Deploy the smart contract:
```
cd dapp/ && truffle deploy
```

Start the api app:
```
cd api/ && npm start
```

Start the client app:
```
cd client/ && npm run dev
```
